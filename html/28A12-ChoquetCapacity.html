Choquet capacity


A Choquet capacity, or just capacity, on a set X is a kind of set function, mapping the power set 𝒫⁢(X) to the real numbers.

Definition.

Let F be a collection of subsets of X. Then, an F-capacity is an increasing set function

I:𝒫⁢(X)→ℝ+

satisfying the following.

  1. 1.

    If (An)n∈ℕ is an increasing sequence of subsets of X then I⁢(An)→I⁢(⋃mAm) as n→∞.

  2. 2.

    If (An)n∈ℕ is a decreasing sequence of subsets of X such that An∈ℱ for each n, then I⁢(An)→I⁢(⋂mAm) as n→∞.

The condition that I is increasing means that I⁢(A)≤I⁢(B) whenever A⊆B. Note that capacities differ from the concepts of measures and outer measures, as no additivity or subadditivity conditions are imposed. However, for any finite measure, there is a corresponding capacity (http://planetmath.org/CapacityGeneratedByAMeasure). An important application to the theory of measures and analytic sets is given by the capacitability theorem.

The (F,I)-capacitable sets are defined as follows. Recall that ℱδ denotes the collection of countable intersections of sets in the paving ℱ.

Definition.

Let I be an F-capacity on a set X. Then a subset A⊆X is (ℱ,I)-capacitable if, for each ϵ>0, there exists a B∈Fδ such that B⊆A and I⁢(B)≥I⁢(A)-ϵ.

Alternatively, such sets are called I-capacitable or, simply, capacitable.
